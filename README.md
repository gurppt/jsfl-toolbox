# JSFL-Toolbox

A growing collection of **JSFL scripts for Adobe Animate // Macromedia Flash**, used in real production contexts.

This repository gathers various tools accumulated over time: some written from scratch, some adapted or modified, and some quite old.  
The goal is simple: centralize useful JSFL utilities that speed up everyday animation work.

The toolbox will evolve over time as new scripts are added.

---
## Included Scripts (current state)

#### -- For animating / moving stuff
##### Autotween
##### ArtBrush
##### Enter Graphic Current Frame
##### GoToLabel
##### GoToKey
##### Graphic Increment / Decrement
##### Convert Timeline to Graphic

#### -- Building helpers for rigging / puppets
##### Copy / Cut / Paste in Place
##### Zero Transform (as-is / as scale)

#### -- Manipulating the timeline
##### Toggle Guide
##### Toggle Outline
##### Toggle Lock
##### Toggle Mask

#### -- Manipulating the library
##### LibAppend
##### Find and Replace

#### -- Batch exporting elements
##### Export Multiple SWF

#### -- Various tools
##### Swap Fill and Stroke
